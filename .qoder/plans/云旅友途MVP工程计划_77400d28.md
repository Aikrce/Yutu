### 1.2 技术栈确认

| 层级 | 技术 | 说明 |
|------|------|------|
| 前端框架 | Vite 5 + React 18 + TypeScript | H5移动端SPA |
| UI样式 | Tailwind CSS 3 + 自定义Design Token | 移动端优先 |
| 状态管理 | Zustand | 轻量级状态管理 |
| 路由 | React Router v6 | 支持Tab式导航 |
| 瀑布流 | 自研虚拟瀑布流组件 | 双列内容流 |
| 地图 | 腾讯地图JavaScript API v2 | LBS匹配 |
| 瀑布流数据 | 真实景点/美食/活动数据 | 搜索填充真实信息 |
| 数据库 | MySQL 8 | 主数据存储 |
| 缓存 | Redis 7 | GeoHash + 会话 |
| 实时通信 | Socket.IO | 私信聊天 |
| 认证 | JWT + 微信OAuth | 多端登录 |
| 文件存储 | 阿里云OSS + CDN | 图片/视频 |

## 二、UI设计规范

### 2.1 设计理念

核心关键词：**温暖、探索、陪伴、信任**

- 温暖：蓝白主色调，圆角设计，友好的视觉氛围
- 品牌：蓝色云朵IP形象，开屏凸显"云旅友途"
- 探索：内容流激发好奇心，地图元素营造探索感
- 陪伴：人物头像突出，社交元素前置
- 信任：认证标识清晰，评价体系透明

### 2.2 色彩系统

**全局主色调（蓝白系）：**
```
主色调（Primary）：#3B82F6（天蓝色 - 品牌主色、按钮、链接）
主色调深（Primary Dark）：#1E40AF（深蓝色 - 强调、选中状态）
主色调浅（Primary Light）：#60A5FA（浅蓝色 - 悬浮、背景）
辅助色（Secondary）：#0EA5E9（青色 - 次级操作、标签）
强调色（Accent）：#F59E0B（琥珀色 - 高亮、价格、重要提示）
成功色（Success）：#10B981（绿色 - 完成状态、在线状态）
警告色（Warning）：#F97316（橙色 - 提醒、紧张状态）
错误色（Error）：#EF4444（红色 - 错误、紧急求助）
```

### 2.3 字体系统

```
标题大（H1）：24px / font-weight: 700
标题中（H2）：20px / font-weight: 600
标题小（H3）：17px / font-weight: 600
正文（Body）：15px / font-weight: 400
辅助文字（Caption）：13px / font-weight: 400
标签（Tag）：11px / font-weight: 500

行高统一：1.5倍字号
```

### 2.4 间距系统

```
基础单位：4px
间距梯度：4 / 8 / 12 / 16 / 20 / 24 / 32 / 40 / 48
页面水平内边距：16px
卡片内边距：12px
组件间距：8px / 12px
模块间距：16px / 24px
```

### 2.5 圆角系统

```
小圆角（标签、小按钮）：8px
中圆角（卡片、输入框）：12px
大圆角（弹窗、底部抽屉）：16px
圆形（头像、图标按钮）：50%
```

### 2.6 阴影系统

```
轻阴影（卡片默认）：0 1px 3px rgba(0,0,0,0.06)
中阴影（卡片悬浮/弹起）：0 4px 12px rgba(0,0,0,0.1)
重阴影（弹窗/浮层）：0 8px 24px rgba(0,0,0,0.15)
```

### 2.7 动效规范

```
快速过渡：150ms ease-out（按钮点击、开关切换）
标准过渡：250ms ease-out（页面切换、卡片展开）
慢速过渡：400ms ease-out（弹窗弹出、抽屉滑出）
下拉刷新：弹性回弹动画
瀑布流加载：渐显淡入（opacity 0→1, translateY 20px→0, 300ms）
匹配动画：脉冲扩散（找搭子/找陪玩时的呼吸灯效果）
```

## 七、移动端H5特殊适配要点

1. **安全区域适配**：使用 `env(safe-area-inset-*)` 适配iPhone刘海屏/底部横条
2. **1px边框**：使用 `transform: scaleY(0.5)` 解决Retina屏1px问题
3. **滚动穿透**：弹窗/抽屉打开时禁止body滚动
4. **软键盘适配**：输入框获焦时，页面自动上推避免遮挡
5. **触摸反馈**：所有可点击元素添加 `active:opacity-70` 触摸反馈
6. **图片懒加载**：使用 Intersection Observer 实现可视区域加载
7. **骨架屏**：所有列表页加载时展示骨架屏，提升感知速度
8. **页面转场**：Tab切换无动画，页面跳转使用 slide-in-right
9. **禁止缩放**：viewport meta 设置 `user-scalable=no`
10. **300ms延迟**：引入 fastclick 或使用 `touch-action: manipulation` 消除点击延迟
